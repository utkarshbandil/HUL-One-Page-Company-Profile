Hindustan Unilever Ltd – One Page Company Profile

This project presents a one-page analytical profile of Hindustan Unilever Limited (HUL) created using Microsoft Excel. The objective is to summarize key financial, ownership, and strategic information about the company in a concise and visually structured format.

The analysis is based on publicly available financial data and qualitative insights gathered from multiple sources such as company annual reports, earnings conference calls, and financial news platforms.

Key Components

- Historical financial metrics (FY21–FY25)
- Key financial ratios
- Share price and volume trends
- Shareholding pattern and top shareholders
- Managerial remuneration analysis
- Capital structure overview
- Recent company developments and updates

Data Sources

- Annual Reports of Hindustan Unilever Limited
- Earnings Conference Call transcripts
- Financial news sources such as The Economic Times and other publicly available reports

Files Included

- HUL_One_Page_Profile.pdf – Final one-page company profile
- HUL_One_Page_Profile_Model.xlsx – Excel model used to build the analysis

Objective

The purpose of this project is to practice financial analysis and present company insights in a concise one-page format commonly used in investment research and corporate analysis.
